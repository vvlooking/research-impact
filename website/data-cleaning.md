# Data Cleaning

Data cleaning involves four steps: (1) combine and clean the datasets in OpenRefine, (2) remove duplicate documents, (3) manually review and edit the dataset, as necessary, and (4) standardize affiliations. Please refer to the next three webpages for detailed instructions on how to clean data.

## Checklist

- [ ] Clean dataset in OpenRefine
- [ ] De-duplicate data
- [ ] Manually review and edit data
- [ ] Standardize affiliations
